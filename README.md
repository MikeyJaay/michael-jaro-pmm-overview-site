# Michael Jaroszynski · PMM Portfolio

[![Live Site](https://img.shields.io/badge/Live%20Site-michael--jaro--pmm--portfolio.netlify.app-blue)](https://michael-jaro-pmm-portfolio.netlify.app/)

[![Site Preview](./public/og-image.png)](https://michael-jaro-pmm-portfolio.netlify.app/)

Portfolio site for Michael Jaroszynski — Technical Product Marketing Manager specializing in AI and enterprise GTM. Built to showcase cross-functional positioning work, career experience, writing, and work samples.

---

## Pages

- **Home** — GTM philosophy, perspectives, and core positioning
- **Career** — Full timeline of roles and companies
- **Writing** — Blog posts and thought leadership
- **Work Samples** — Artifacts and deliverables

---

## Tech Stack

React 18 · TypeScript · Vite · Tailwind CSS · Framer Motion · Radix UI · React Router

---

## Local Dev

```bash
npm install
npm run dev
```

Runs at `http://localhost:8080`
